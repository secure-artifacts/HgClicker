# HeyGen Helper T2 / Auto Click

**当前版本：2.3.0**

---

## v2.3.0 更新内容

### 🎬 字幕功能全面修复

- **修复 captionId 无法保存**：HeyGen 当前版本不在 DOM 上暴露字幕样式的选中状态（aria、class、style 均无差异），原有 7 种检测策略全部失效。改用**点击拦截**方案：在 document capture 阶段监听 `[data-caption-id]` 卡片点击，用户点选哪张样式即记录其 id，保存时直接使用。
- **修复字幕样式无法自动应用**：`_applySettings()` 中查找样式卡片的 XPath（`h2[text()="Captions"]`）在当前 HeyGen DOM 中不存在，导致卡片从未被点击。改为直接使用 `querySelector('[data-caption-id="xxx"]')`，并等待卡片实际渲染后再点击。
- **修复位置 Y 无法自动应用**：同一 XPath 问题导致 positionY 设置入口每次等待 5 秒后失败。已移除失效的容器守卫，直接查找 Y 输入框。
- **修复字号无法自动应用**：`_setFontSize()` 依赖同一失效 XPath 作为容器，导致字号从未被设置。改为在 `document.body` 范围内查找字号元素。
- **优化保存响应速度**：已有点击记录时跳过面板打开和卡片等待流程（原最多等待 8 秒），保存操作现在立即执行。

- **设置按钮移至图标右侧**：⚙ 设置按钮调整至插件图标的右侧。

## v2.2.0 更新内容（相比 v2.0.0）

### 🐛 崩溃修复
- **修复 Chrome 浏览器崩溃问题**：`browsingData.remove()` 使用了 `serviceWorkers`、`cacheStorage`、`fileSystems` 等危险类型，触发 Chrome 内部 use-after-free（SIGSEGV at 0x37）。已限制为仅清理 Cookies、LocalStorage、IndexedDB、Downloads，彻底消除崩溃风险。

---

### 🎬 字幕自动设置（全面重构）

- **自动应用上次字幕预设**：首次手动配置字幕后，后续打开编辑器会自动应用上次保存的设置（样式、位置 Y、字号），无需每次手动操作。
- **新增 Script 输入触发**：除页面加载外，在 Script 区域粘贴或输入文字时也会触发字幕自动设置，触发时机更可靠（页面必然已渲染完成）。
- **修复触发条件**：原触发条件依赖页面内文字 `Delivery style`（仅在特定面板展开时才出现），导致字幕功能从未被触发。已改为 URL 路径匹配（`/create-v4/`），稳定触发。
- **修复语法错误**：代码中存在 `const url` 重复声明（SyntaxError），导致整个 Content Script 无法加载，所有自动化功能全部失效。已修复。
- **重试机制**：若因页面渲染延迟导致字幕设置失败，最多自动重试 3 次；成功后不再重试，避免重复操作。
- **并发保护**：新增内部锁（`_isApplying`），防止页面加载和 Script 输入两个触发点同时执行。
- **Generate 按钮保存预设**：点击 Generate 时自动保存当前字幕配置（不只是 Ctrl+B 路径），确保预设始终与最新操作同步。

---

### ⬇️ 批量下载（稳定性提升）

- **修复漏下载**：原先在页面开始时一次性收集所有卡片 DOM 引用，React 重新渲染后引用断开（`isConnected = false`），导致部分视频被跳过。现改为每轮迭代重新按唯一标识查找卡片。
- **修复重复下载**：卡片嵌套检测偶尔失效，同一张卡片被收集两次导致重复下载。现使用视频标题作为唯一 Key 进行去重。
- **修复小窗口漏下载**：窗口较小时 HeyGen 可能启用虚拟滚动，部分卡片不在 DOM 中。下载前新增自动滚动到底部流程，强制加载全部卡片后再开始处理。
- **下载与删除分离**：批量下载完成后不再自动触发删除操作，删除功能已独立移至「清理数据」对话框，避免误操作。

---

### 🖼️ 上传头像修复

- **适配 HeyGen 新 UI**：HeyGen 将「Clone a real person」由按钮改为 H1 标题，「Upload photo」按钮仅在悬停「Create a virtual character」卡片时出现。已重写点击逻辑：定位卡片图片 → 触发 hover 事件 → 等待按钮出现后点击。
- **修复从其他页面跳转后弹框不出现**：原等待时间不足，改为轮询等待卡片加载完成（最长 10 秒），再执行点击。

---

### 🎨 界面优化

- **设置面板重设计**：白色背景、柔和阴影、分组标题与分隔线、选中项高亮蓝色调，风格统一整洁。
- **清理数据对话框重设计**：只展示将被清理的项目，保留项不再显示；新增红色警告区和绿色保留说明区，按钮样式统一。
- **工具栏自动隐藏**：延迟由 2 秒调整为 3 秒，操作更从容。
- **新增「自动填写姓名」开关**：设置面板新增 `autoFillNameEnabled` 开关。

---

### 🔖 插件图标更新

- 替换为全新彩色 Pinwheel 图标，白色背景处理为透明，分辨率覆盖 16 × 16、48 × 48、128 × 128。

## 安装方式

1. 打开 `chrome://extensions`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本目录
4. 访问 `https://app.heygen.com` 即可使用
